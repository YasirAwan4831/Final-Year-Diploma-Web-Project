# 🎓 Final-Year-Diploma-Web-Project

Welcome to the official repository of our **Final Year Diploma Web Development Project**.  
This project has been created as part of the **Final Year Information Technology Diploma** final Year submission at **Red-C College**.

It showcases the complete skills learned during the diploma — including **HTML5**, **CSS3**, **JavaScript and Other**  combined to create a visually appealing, responsive and professional website.

---

##  Project Overview

The **Final Year Diploma Web Development Project** is a fully functional, user-friendly, and visually engaging website developed as a business prototype for an online digital agency. It showcases modern design principles, interactive elements and a structured layout aimed at providing digital services in a professional manner.

**The project includes multiple sections such as:**
- **Home Section** with parallax scrolling  
- **Services Section** displaying provided offerings  
- **Portfolio Section** showcasing previous works or projects  
- **Team Section** introducing developers and collaborators  
- **Contact Section** for user interaction and feedback  

> The website is fully responsive, meaning it adapts smoothly to desktop, tablet and mobile screens.

##  Project Objectives

The **Final-Year-Diploma-Web-Project** was designed with the following primary objectives:

1. **Educational Purpose:**  
   To demonstrate the understanding of modern web development principles learned during the IT program.

2. **Responsive Design Implementation:**  
   To build a fully responsive layout compatible with all devices  desktop, tablet and mobile.

3. **Interactive User Experience:**  
   To integrate smooth animations, parallax scrolling and hover effects that engage visitors visually.

4. **Code Reusability & Structure:**  
   To maintain clean, modular, and well-documented HTML, CSS, JavaScript and JSON code that follows best practices.

5. **Professional Presentation:**  
   To create a real world level website suitable for showcasing in a professional portfolio or online resume.

---

##  Key Features

The website includes a variety of modern web features and design elements:

- 🌐 **Multi-Page Structure:**  
  Home, Services, Portfolio, Team and Contact Us pages organized cleanly within folders.

- 💫 **Parallax & Scroll Animations:**  
  Visually appealing scroll effects using JavaScript and CSS transitions.

- 📱 **Fully Responsive Design:**  
  Uses flexible layouts and media queries to ensure perfect display on all screen sizes.

-  **Custom Styling & Fonts:**  
  Integrated Poppins font and global CSS variables for consistent theme and typography.

-  **Reusable Components:**  
  Buttons, cards and utility classes reused across multiple pages for efficiency.

- 📸 **Image-Based Sections:**  
  High-quality visuals for Services, Portfolio and Team members.

-  **Optimized Folder Structure:**  
  Organized assets (CSS, JS, Images, Data, Fonts) for easy maintenance and scalability.

- 📧 **Contact Form:**  
  Clean and user friendly contact form with form-validation prepared in JavaScript.

## ⚙️ Step 5: Installation & Usage Instructions

To run and explore the **Final-Year-Diploma-Web-Project** on your local system, follow the steps below carefully.  
These instructions are designed to help new users, classmates, or evaluators easily set up and test the project without any confusion.

---

###  1. Prerequisites

Before starting, make sure you have the following installed:

-  **Web Browser:** Google Chrome or Edge  
- 💻 **Code Editor:** Visual Studio Code (recommended)  
- 🌐 **Live Server Extension:** For running the project locally  
- ⚙️ *(Optional)*: Node.js (if you plan to extend or deploy it later)

---

###  2. Download or Clone the Repository

You can get the project in two ways:

#### Option 1 — Download ZIP
1. Click the green **Code** button on the repository page.  
2. Select **Download ZIP**.  
3. Extract the ZIP file into your desired folder.

#### Option 2 — Clone using Git
If you have Git installed:
```bash
git clone https://github.com/YasirAwan/Final-Year-Diploma-Web-Project.git
```
#### 3. Open the Project in VS Code
1. Launch **Visual Studio Code** 
2. SleGo to **File → Open Folder**  
3. Select the extracted folde
4. Open the file **index.html**


####  4. Run the Project (Live Server)
To view your website locally:
1. Right-click on **index.html** 
2. Select ***“Open with Live Server”**  
3. Your browser will open automatically and display the homepage.

###  5. Project Navigation

Once the project runs successfully, you can explore the following sections:

- **Home Section:** Animated intro with parallax effect  and footer details
- **About Section:** Overview of the project's purpose  
- **Services Section:** Displays data dynamically from `services.json`  
- **Team Section:** Shows team information from `team.json`  
- **Contact Section:** Includes contact form and footer details  

###  6. Optional: Modify Data

To update your content dynamically, open the **data** folder and edit:

- `site-info.json` → Change project title or description  
- `services.json` → Add/remove services  
- `team.json` → Update team details (name, role, image)  

Changes will automatically appear on the site when refreshed.
